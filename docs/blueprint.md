# **App Name**: Viewora AI Coach

## Core Features:

- User Authentication: Secure user authentication via Google and Microsoft OAuth2. Automatic profile creation on first login.
- Token Economy: In-app currency using tokens. Users spend tokens for AI photo analysis.
- Payment System: Integrated payment system (Iyzico/Stripe) to purchase token packages, updating token balance upon successful payment.
- Welcome Bonus: Automated welcome gift of 10 tokens to new users after one hour of registration, accompanied by an in-app notification.
- User Tier System: Gamified user tiers (Basic, Mid, Pro) with escalating benefits based on activity and token purchases.
- AI Photo Coaching: AI-powered photo analysis using Vision API (GPT-4o/Claude 3.5 Vision) providing feedback on Light, Composition, Emotion, and Technique. LLM is used as a tool to provide 3 concrete improvement tips.
- Photo Gallery: Personalized 'Art Gallery' (archive) where users can view their previously analyzed photos.

## Style Guidelines:

- Primary color: Soft teal (#A0E9D3), reflecting creativity and growth, and inspired by the cool yet vibrant tones of professional photography environments.
- Background color: Light gray (#F0F0F0), a neutral backdrop allowing the vibrant photography and UI elements to pop.
- Accent color: Warm gold (#FFD700), used sparingly to highlight key interactive elements and calls to action, drawing from classic photographic printing techniques.
- Headline font: 'Belleza' sans-serif, aligned to fashion, art, and design. Body font: 'Alegreya' serif for longer text.
- Clean, minimalist icons to represent different coaching aspects (e.g., composition, lighting), providing a visual guide to the AI feedback.
- Clean and structured layout, with a focus on showcasing user photos prominently. Use of card-based UI for lessons and analysis results.
- Subtle transitions and animations to enhance user experience. Feedback animations on AI analysis for an engaging feel.
